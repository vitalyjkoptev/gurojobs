# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it responsibly:

- Email: gurojobs@gmail.com
- Do NOT open a public GitHub issue for security vulnerabilities

## Sensitive Data

This repository does not contain:
- API keys or tokens
- Database credentials
- Server passwords or SSH keys
- Firebase service accounts

All sensitive configuration is managed via `.env` files (not tracked in git).

## Environment Setup

Copy `.env.example.gurojobs` to `.env` and fill in your own credentials.
